Подсказки по гиту git init создает репазиторий в папке git add имяфайла.md следит за файлом git status посмотреть статус git commit -m "название комита" git log показывает все коммиты подробно git log --oneline показывает все коммиты упрощенно git commit --amend -m "error" удаляет предыдущий комит и создает новый для всех последних файлов git add -A добавить все файлы для отслеж. git log --oneline --all показывает все ветки даже если находится в другой git checkout 726ae1f перейти в ветку git checkout main вернутся в main
