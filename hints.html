Подсказки по гиту git init создает репазиторий в папке git add имяфайла.md следит за файлом git status посмотреть статус git commit -m "название комита" git log показывает все коммиты подробно git log --oneline показывает все коммиты упрощенно git commit --amend -m "error" удаляет предыдущий комит и создает новый
